# Student Performance Prediction using Machine Learning

## 📌 Project Description

This project predicts student performance using Machine Learning techniques.
The model analyzes student academic data and predicts whether the student will perform well based on different features such as exam scores.

The project was implemented using Python in Jupyter Notebook.

---

## 🎯 Objective

The objective of this project is to develop a machine learning model that can analyze student academic performance and make predictions based on available data.

---

## 📂 Dataset

The dataset contains student academic details such as:

* Math Score
* Reading Score
* Writing Score
* Gender
* Parental Education
* Lunch Type
* Test Preparation Course

The dataset is used to train a machine learning model to predict student performance.

---

## ⚙️ Technologies Used

* Python
* Jupyter Notebook
* Pandas
* NumPy
* Matplotlib
* Scikit-learn

---

## 🧠 Machine Learning Algorithm

The project uses the following algorithm:

* Logistic Regression

The model is trained using student score data to predict performance.

---

## 🔄 Project Workflow

1. Import libraries
2. Load dataset
3. Data preprocessing
4. Feature selection
5. Train-test split
6. Model training
7. Prediction
8. Model evaluation

---

## 📊 Output

The machine learning model predicts student performance based on academic scores and provides accuracy results.
